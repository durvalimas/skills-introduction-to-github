## Suporte Rede TCP/IP

Este perfil documenta recursos e suporte para redes TCP/IP.

### Protocolos Suportados

- **TCP (Transmission Control Protocol)** – protocolo orientado a conexão para transmissão confiável de dados
- **IP (Internet Protocol)** – protocolo responsável pelo endereçamento e roteamento de pacotes
- **UDP (User Datagram Protocol)** – protocolo sem conexão para transmissão rápida de dados
- **ICMP (Internet Control Message Protocol)** – protocolo de diagnóstico e controle de rede

### Recursos

- Configuração e gerenciamento de interfaces de rede
- Diagnóstico de conectividade (ping, traceroute)
- Resolução de nomes (DNS)
- Segurança de rede e firewall
